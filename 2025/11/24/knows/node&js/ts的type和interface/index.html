总结

概念 说明
Type TypeScript 类型别名，定义自定义类型
用途 简化复杂类型、提高代码可读性
特点 支持联合、交叉、映射等高级特性
vs Interface Type 更灵活，Interface 更适合对象
编译后 Type 在编译后被完全删除，只在开发时存在

Type 是纯开发时的类型定义，运行时不存在。 🚀

总结

概念 说明
T 泛型占位符，代表待定类型
用途 让类型更灵活，避免重复代码
声明 <T> 在尖括号中声明
使用 Type<T> 或 Function<T>()
约束 <T extends Type> 限制 T 的范围
多参 <T, U, K> 支持多个泛型参数

T 让你写出更通用、更可复用的类型和函数！ 🚀
